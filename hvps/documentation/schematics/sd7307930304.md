## Description

Phase control thyristor stack circuit schematic. Each stack contains 14 Powerex T8K7 thyristors in series for voltage holdoff.

         +---->| SCR   |               |

         |          ...                     (repeat x14)

                        |
                        v
                  To Filter Inductor

  Gate Drive: Fiber optic from controller
  First pulse: 240V, 16us
  Subsequent: 120V, 16us, 15-degree bursts

  Voltage Sharing:

  - ~20 Mohm resistance per thyristor (parallel path)
  - Total stack resistance: ~280 Mohm
