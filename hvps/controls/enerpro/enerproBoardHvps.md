## Enerpro Modifications for SLAC HVPS

## Introduction

The SLAC accelerator SPEAR3 has a twelve-pulse, thyristor controlled, high voltage power supply (HVPS).  We are in the process of an upgrade of our HVPS controller, leaving the high-power part of the HVPS as is.  As part of this upgrade, we want to ensure that we use a modern circuit to generate the thyristor triggers for the supply and that we properly interface the rest of our controller to the trigger circuit.  This note documents our current controller and asks Enerpro for some detailed information and advice so that we can properly interface our controller with the Enerpro firing circuit.

We feed the SIGHI input of the Enerpro board from an external controller, which we will also redesign and rebuild during our upgrade project.  This controller consists of two parts, a programmed logic controller (PLC) and an analog regulator board.  A slow external software control system determines the target voltage of the HVPS, which it sends to the PLC.  From the target voltage, the PLC generates two analog output values.  One is an approximate value it calculates for SIGHI.  The second is the analog signal that it scales to be the reference for the analog regulator.  The analog regulator compares the readback of the HVPS with the reference and generates an error signal, which is weighted and added to the PLC analog output.  The SIGHI input to the Enerpro is the sum of these two signals.  Figure 4 shows a filtered version of the error signal.  Power line harmonics are clearly visible on the error signal.
